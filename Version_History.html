Version 3.0

Batch Processing Error Resolution & Stability Improvements

v3.0

Batch Processing Error Resolution & Enhanced Stability

January 2025

Version 3.0 focuses on resolving critical batch processing errors that were preventing successful completion of multi-video workflows. This release addresses various edge cases, error handling issues, and stability problems that could cause batch operations to fail or hang indefinitely.

Major Fix: Batch processing now handles errors gracefully and continues processing remaining videos in the queue without stopping the entire operation.

Critical Bug Fixes

  • Batch Queue Hanging: Fixed issue where batch processing would hang indefinitely when encountering corrupted or unsupported video files
  • Memory Leak in Batch Mode: Resolved memory accumulation during long batch processing sessions that could cause system crashes
  • Error Propagation: Fixed cascading errors where one failed video would cause subsequent videos in the batch to fail
  • Progress Tracking Issues: Corrected batch progress reporting that would show incorrect completion percentages
  • Temp File Cleanup: Fixed incomplete cleanup of temporary files during batch processing failures

Stability Improvements

  • Error Recovery: Enhanced error recovery mechanisms that allow batch processing to continue after individual video failures
  • Resource Management: Better GPU memory management during batch operations to prevent out-of-memory errors
  • File Validation: Added pre-processing validation to identify problematic files before starting batch operations
  • Logging Enhancement: Improved error logging and reporting for better debugging of batch processing issues

New Features

  • Batch Error Summary: Added comprehensive error summary report at the end of batch processing sessions
  • Skip Failed Videos Option: New option to automatically skip videos that fail initial validation checks
  • Retry Mechanism: Automatic retry functionality for transient errors during batch processing

Technical Changes

  • Exception Handling: Improved exception handling throughout the batch processing pipeline
  • Resource Cleanup: Enhanced automatic cleanup of resources and temporary files on error conditions
  • Thread Safety: Improved thread safety in batch processing operations to prevent race conditions
  • Status Monitoring: Enhanced real-time status monitoring and progress tracking for batch operations
Note: This update maintains full backward compatibility. Existing batch processing configurations will continue to work with improved reliability and error handling.

Version 2.0

Enhanced MKV Support & Improved Video Parsing

v2.0

Enhanced MKV Support & Robust Video Parsing

December 2024

Version 2.0 focuses on improving video file compatibility, particularly for MKV files, and enhancing the overall robustness of video parsing throughout the application. This release addresses issues where MKV files were not being properly recognized and parsed, leading to incorrect frame counts and duration information.

Major Improvement: MKV files now parse correctly with accurate frame counts, duration, and metadata extraction.

New Features

  • Enhanced MKV Support: Improved parsing for Matroska video files with proper handling of 'N/A' values in metadata fields
  • Robust Video Info Extraction: Multiple fallback mechanisms for video parsing when primary methods fail
  • OpenCV Integration: Automatic fallback to OpenCV for video information when ffprobe encounters parsing issues

Improvements

  • Video Parsing Robustness: Enhanced error handling for duration, frame count, and bitrate parsing from ffprobe output
  • Duration Calculation: Improved accuracy of video duration calculation using multiple methods and cross-validation
  • Frame Count Detection: Better frame count detection with fallback to direct frame counting when needed
  • Error Reporting: More detailed logging and error messages for video parsing issues

Bug Fixes

  • MKV Duration Parsing: Fixed "could not convert string to float: 'N/A'" error when parsing MKV file durations
  • Zero Frame Count Issue: Resolved issue where MKV files showed 0 frames and 0.000s duration
  • Bitrate Parsing: Fixed parsing errors for video bitrate information in various container formats
  • Format Detection: Improved format detection and handling for edge cases in video containers

Technical Changes

  • Enhanced get_video_info_fast(): Added 'N/A' value handling and improved error recovery
  • Enhanced get_video_info(): Improved comprehensive video information extraction with better fallbacks
  • OpenCV Fallback Integration: Automatic fallback to OpenCV for missing video information
  • Duration Calculation Logic: Enhanced duration calculation using frame count and FPS when direct parsing fails
Note: This update maintains full backward compatibility. All existing functionality remains unchanged, with only improvements to video parsing robustness.

Version 1.0

Initial Release - Complete Video Upscaling Suite

v1.0

Initial App Release

December 2024

Version 1.0 represents the initial release of the SECourses Video Upscaler Pro, providing a comprehensive suite of video upscaling tools with multiple AI models and advanced processing capabilities.

Core Features

  • STAR AI Temporal Upscaling: Advanced AI-powered video upscaling with prompt guidance and temporal consistency
  • Image-Based Super Resolution: Multiple RealESRGAN, DAT, and SPAN models for fast spatial upscaling
  • Auto-Captioning with CogVLM2: AI-powered video description generation with multiple quantization options
  • Face Restoration (CodeFormer): Advanced face enhancement and restoration capabilities
  • RIFE Frame Interpolation: AI-powered frame interpolation for smooth motion and higher frame rates
  • Batch Processing: Automated processing of multiple videos with smart prompt handling
  • Video Editor & Cutting: Precise video cutting and segment extraction tools
  • Scene-Based Processing: AI-powered scene detection and per-scene processing

Advanced Features

  • Multi-GPU Support: Automatic GPU detection and selection for optimal performance
  • VRAM Optimization: Advanced memory management with configurable chunk sizes
  • FFmpeg Integration: Hardware-accelerated encoding with NVIDIA NVENC support
  • Preset System: Save and load processing configurations for consistent results
  • Comparison Video Generation: Automatic side-by-side and top-bottom comparison videos
  • Metadata Management: Comprehensive logging and metadata saving for reproducibility

Supported Formats

  • Video Input: MP4, AVI, MOV, MKV, WebM, WMV, FLV, M4V
  • Image Input: JPG, PNG, BMP, TIFF for direct image upscaling
  • Frame Sequences: Support for frame folder processing and conversion
Initial Release: Version 1.0 established the foundation for all subsequent improvements and feature additions.
